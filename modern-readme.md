## Modern Install Script

### What Each Command Does:

- `sudo apt update`: Updates available package information.
- `sudo apt upgrade -y`: Installs the latest versions of the installed packages.
- `neofetch`: Shows a summary of your systemâ€™s information.
- `htop`: Displays an interactive process viewer.
- `btop`: Another system resource monitor written in Python.
- `tree`: Displays the directory structure as a tree.
- `tmux`: Terminal multiplexer to manage multiple terminal windows.
- `unzip`: Extracts files from a ZIP archive.
- `curl`: Transfers data over the network from URLs.
- `wget`: Downloads files from the web.
- `git`: Version control tool for managing code.
- `nano`: Easy-to-use terminal text editor.
- `vim`: A powerful text editor.
- `neovim`: A modern version of Vim with additional features.
- `zsh`: A powerful shell (alternative to Bash).
- `fzf`: A command-line fuzzy finder to search files or directories.
- `bat`: A better `cat` command with syntax highlighting.
- `exa`: A better `ls` for listing files.
- `ripgrep`: A faster alternative to `grep`.
- `fd-find`: A faster and simpler alternative to `find`.
- `jq`: Command-line tool to parse and manipulate JSON.
- `build-essential`: Installs the essential development tools.
- `python3`: Python 3 interpreter.
- `python3-pip`: Python's package installer.
- `gcc`: C compiler.
- `make`: Tool to automate building programs from source.
- `cmake`: A tool for building, testing, and packaging software.
- `ranger`: Terminal-based file manager.
