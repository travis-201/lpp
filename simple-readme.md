## Simple Install Script

### What Each Command Does:

- `sudo apt update`: Updates the list of available packages and their versions.
- `sudo apt upgrade -y`: Upgrades installed packages to their latest versions.
- `neofetch`: Displays system information in the terminal.
- `htop`: Interactive process viewer to monitor system processes.
- `tree`: Displays directory structure as a tree.
- `tmux`: Terminal multiplexer that allows multiple terminal sessions.
- `unzip`: Extracts ZIP archives.
- `curl`: Command-line tool for transferring data with URLs.
- `wget`: Command-line tool to download files from the web.
- `git`: Version control system for tracking changes in files and collaborating.
- `nano`: Simple terminal text editor.
- `vim`: Advanced text editor for code and general editing.
- `build-essential`: Installs essential development tools (compilers, make, etc.).
- `python3`: Python 3 programming language interpreter.
- `python3-pip`: Python package manager for installing Python libraries.
- `gcc`: GNU C compiler.
- `make`: Tool to automate the process of compiling software.
